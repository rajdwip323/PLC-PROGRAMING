# PLC-Based Motor/Pump Control System

## 📌 Overview
This project is a basic PLC-based motor/pump control system developed using Siemens LOGO! Soft Comfort.  
It demonstrates simple industrial automation logic for controlling a motor using start and stop switches.

## ⚙️ Features
- Start/Stop control of motor
- Safe operation using basic logic
- Simple ladder logic implementation
- Suitable for beginners in PLC automation

## 🛠️ Tools Used
- Siemens LOGO! Soft Comfort
- Ladder Logic Programming

## 🔌 Working Principle
- When the START switch is pressed, the motor turns ON  
- When the STOP switch is pressed, the motor turns OFF  
- The system uses basic PLC logic to control output (Q1)

## 📷 Project Structure
- Input: Switches (Start/Stop)
- Output: Motor (Q1)
- Logic: Ladder Diagram

## 🎯 Learning Outcome
- Basic understanding of PLC programming  
- Practical implementation of motor control logic  
- Hands-on experience with industrial automation concepts

## 🚀 Future Improvements
- Add timer (TON) for delay control  
- Add overload protection logic
